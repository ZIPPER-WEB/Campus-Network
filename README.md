# VVNA College Campus Network Project 🎓🌐

## 📌 Overview
This project presents a comprehensive, scalable, and secure campus network architecture designed specifically for **VVNA College**. It aims to provide seamless connectivity across different academic departments, administrative buildings, and student facilities while ensuring high availability and robust security.

## ✨ Key Features
* **Hierarchical Network Design:** Utilizes Core, Distribution, and Access layers for optimal traffic flow and scalability.
* **VLAN Segmentation:** Traffic isolation for Students, Faculty, Administration, and Guests to improve security and broadcast efficiency.
* **Redundancy & High Availability:** Implementation of backup links and protocols (e.g., HSRP/VRRP, Spanning Tree) to prevent network downtime.
* **Security Policies:** Access Control Lists (ACLs) and firewall rules to protect sensitive college data and administrative networks.
* **Wireless Connectivity:** WLC-managed enterprise Wi-Fi coverage across the campus and library.

## 🛠️ Technologies & Tools Used
* **Simulation Software:** Cisco Packet Tracer / GNS3 *(update as needed)*
* **Routing Protocols:** OSPF / EIGRP *(update as needed)*
* **Switching Concepts:** VLANs, VTP, STP, Inter-VLAN Routing
* **IP Addressing:** IPv4/IPv6 Subnetting & DHCP Configuration
